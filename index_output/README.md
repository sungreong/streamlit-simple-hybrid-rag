# Index Output Directory

이 폴더에는 사전 빌드된 검색 인덱스 파일이 포함되어 있습니다.

## 포함된 파일

- `metadata.json` - 문서 메타데이터 및 청크 정보
- `bm25.pkl` - BM25 검색 인덱스
- `index.faiss` - FAISS 벡터 검색 인덱스

## 사용 방법

이 인덱스 파일들은 샘플 데이터(`data/` 폴더)를 기반으로 미리 생성되었습니다.
앱을 실행하면 바로 검색을 시작할 수 있습니다.

## 새 문서 추가 시

새로운 문서를 추가하려면:

1. `data/` 폴더에 `.txt` 또는 `.md` 파일 추가
2. `python vectorize.py` 실행하여 인덱스 재생성
3. 앱 재시작

## 주의사항

⚠️ 인덱스 파일은 용량이 클 수 있습니다 (수백 MB).
⚠️ 민감한 정보가 포함된 문서로 인덱싱한 경우, 이 폴더를 공개 저장소에 올리지 마세요.
